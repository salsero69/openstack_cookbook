Welcome to GitHub Pages.

This is my OpenStack GitHub repository which contains a number Bash shell scripts to make the management of OpenStack an easier task while learning its ins and outs. You're more than welcome to pull the code and use it.

If you're interested in adding some of your own scripts or have comments, critiques for the scripts that currently exist, please contact me.

Currently most of what's in here is based on DevStack pre-IceHouse release. So some of the scripts may break going forward. As I learn more about OpenStack, I'm hoping to have an IceHouse release set of scripts that I can use consistently.
